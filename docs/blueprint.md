# **App Name**: Dulce Sabor Admin Panel

## Core Features:

- Authentication and Roles: Implement secure authentication with Firebase Auth, ensuring only users with the 'admin' role can access the admin panel.
- Inventory Management: Enable adding, editing, and deleting products in the inventory, with form validation and image preview.
- Real-time Order Management: Provide real-time updates on orders, allowing filtering by status, detailed view, and status changes.
- Dynamic Dashboard: Generative AI tool which responds to user requests in the dashboard and can generate sales reports
- Data Storage: All the details of users, orders and products are safely stored and retrieved using Cloud Firestore

## Style Guidelines:

- Primary color: Soft pink (#F9CADA) to evoke sweetness and delicacy.
- Background color: Light cream (#FFF4E6) to create a warm, inviting feel.
- Accent color: Muted brown (#D6A77A) to add a touch of elegance and sophistication.
- Headline font: 'Poppins' (sans-serif) for titles, providing a modern touch.
- Body text font: 'Inter' (sans-serif) for clear and readable content.
- Use simple, consistent icons for the sidebar and throughout the dashboard.
- Modern dashboard layout with a fixed sidebar, topbar, and content area.
- Subtle fade-in animations for lists and smooth transitions on hover.
